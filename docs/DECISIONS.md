# DECISIONS - what'shappeing

## Escolha por Python

- Linguagem madura com excelente suporte a desktop
- Ecossistema rico para aplicações locais
- Fácil empacotamento com PyInstaller

## Escolha por CustomTkinter

- Interface moderna e profissional
- Baseado em Tkinter, nativo do Python
- Aparência consistente com Windows
- Leve e sem dependências externas pesadas

## Escolha por SQLite

- Banco local, sem servidor
- Persistência confiável e rápida
- Não requer instalação adicional
- Suporte nativo em Python

## Escolha por PyInstaller

- Ferramenta padrão para empacotamento Python
- Gera executáveis standalone
- Suporte completo a Windows
- Configuração simples

## Escolha por Inno Setup

- Criador de instaladores profissional para Windows
- Gratuito e confiável
- Suporte a atalhos e desinstalação
- Integração fácil com scripts

## Escolha por Jenkins

- Servidor de CI/CD maduro
- Suporte nativo a pipelines declarativas
- Agentes Windows para builds desktop
- Integração com GitHub

## Motivo de Manter Estrutura Enxuta

- Projeto desktop local, não requer arquitetura complexa
- Foco em funcionalidade sobre estrutura
- Manutenibilidade e velocidade de desenvolvimento
- Evitar overhead desnecessário
